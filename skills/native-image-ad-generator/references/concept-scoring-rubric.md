# Concept Scoring Rubric

Score each concept 1-5 on five dimensions (max 25).

## Dimension 1: Emotional Resonance
Does the concept tap into real emotional drivers from research?

| 5 | Directly addresses #1-2 driver. Mirrors real customer quotes. Target feels genuinely understood. |
| 4 | Addresses top 3-4 driver. Clear customer voice connection. |
| 3 | Real driver but generic — could apply to many products. |
| 2 | Feature-focused, not feeling-focused. |
| 1 | No emotional resonance. Purely functional. |

## Dimension 2: Scroll-Stop Power
Would this interrupt scrolling in under 1 second?

| 5 | Visually arresting — high contrast, unexpected, stands out among other ads. |
| 4 | Strong visual hook. Clear focal point. Noticeably above average. |
| 3 | Competent but nothing unexpected. Blends in. |
| 2 | Generic. Could be any brand. |
| 1 | Scrolled past without a glance. |

## Dimension 3: Brand Alignment
Does visual style, tone, and message feel on-brand?

| 5 | Could post on brand's IG tomorrow. Perfect match. |
| 4 | Strong fit, minor adjustments needed. |
| 3 | Generally in territory but some elements off. |
| 2 | Noticeable disconnect — right category, wrong brand. |
| 1 | Completely off-brand. |

## Dimension 4: Strategic Differentiation
Exploits competitive gaps from research?

| 5 | Directly fills gap. No competitor runs this angle. |
| 4 | Competitors weak here. Brand's positioning makes it work better. |
| 3 | Good angle but competitors also use it. |
| 2 | Common category angle, no unique positioning. |
| 1 | Copies competitor approach directly. |

## Dimension 5: Prompt Feasibility
Can Gemini realistically generate this at ad-quality?

| 5 | Simple, well-defined. Product photography, lifestyle, mood. Minimal text. High confidence. |
| 4 | Achievable with careful prompting. 1-2 short text elements. Good odds. |
| 3 | Possible but may need attempts. Some tricky elements (hands, specific text). |
| 2 | Challenging. Perfect text, specific expressions, complex arrangements. |
| 1 | Very unlikely. Long text, logos, celebrity likenesses. |

## Selection Rules

1. Rank by total score
2. **Diversity:** top 5 must include ≥3 different ad types
3. **Funnel coverage:** ≥1 each for awareness, consideration, conversion
4. **Tiebreaker:** prefer higher Scroll-Stop Power
