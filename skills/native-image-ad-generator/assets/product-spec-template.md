# Product Spec Sheet Template

Extract from brand research dossier before writing NanoBanana prompts. Ensures every prompt describes the product accurately.

```
PRODUCT SPEC SHEET
==================
Product Name: [Full name as on brand website]

Physical Description: [2-3 sentences — shape, dimensions, proportions,
notable features. Describe to someone who has never seen it.]

Color(s): [Detailed with hex codes if available. All visible colors —
body, lid, sleeve, interior. "soft sage green with fine speckled texture"
not just "green"]

Key Visual Features:
- [e.g., "speckled matte sage green glaze"]
- [e.g., "cream silicone slip-on lid with dome top"]
- [e.g., "no handle — clean modern silhouette"]

Scale Reference: [Size relative to common objects. Fits in cupholder?
Held in one hand?]

Accessories/Components Visible:
- [e.g., "cream silicone lid (slip-on)"]
- [Note which can be shown with/without]

What It Must NOT Look Like:
- Must NOT [common AI hallucination]
- Must NOT [competitor confusion]
- Must NOT [wrong feature/finish/pattern/accessory]

Brand Colors:
- Primary: [Name] (#HEX)
- Secondary: [Name] (#HEX)
- Accent: [Name] (#HEX)
- Background: [Name] (#HEX)
- Text: [Name] (#HEX)

Price Point: [For offer/comparison ads]

Key Proof Points (for text overlays):
- [e.g., "4.9 stars from 236 reviews"]
- [e.g., "Handmade in studio"]
```

## Usage
1. Fill from research dossier
2. Keep accessible while writing prompts
3. Copy-paste physical description + "Must NOT" into each prompt
4. Adjust per concept (with/without sleeve, held vs. on surface, etc.)
